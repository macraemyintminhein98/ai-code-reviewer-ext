# AI Code Reviewer Chrome Extension

**Get instant, AI-powered code review suggestions directly in your browser!**

This lightweight Chrome Extension is designed for junior to mid-level web developers, bootcamp students, and freelancers who want quick feedback on their HTML, CSS, and JavaScript code. It identifies typos, common anti-patterns, and suggests simple style fixes, helping you write cleaner, more efficient code.

## Features:

*   **Instant Code Review:** Basic suggestions for HTML, CSS, and JavaScript.
*   **Browser Integration:** Works seamlessly in popular online IDEs, GitHub Gist, and repository pages.
*   **AI-Powered:** Leverages advanced AI for smart and relevant feedback.
*   **Easy to Use:** Get suggestions with a click or on code selection.

## How it Works:

1.  **Select Code:** Highlight a code snippet in your online editor or GitHub page.
2.  **Get Suggestions:** The extension sends the code to a secure, serverless AI endpoint.
3.  **See Feedback:** Instant review suggestions appear directly on the page.
